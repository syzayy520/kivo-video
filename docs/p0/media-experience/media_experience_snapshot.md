# Media Experience Snapshot

MediaExperienceSnapshot connects playback core contracts to the private-cinema experience.

## Scope

It carries title, artwork refs, collection refs, progress, watched state, resume point, version summary, source availability, subtitle availability, audio and video quality summaries, quality badges, metadata provenance, and manual override state.

## Badge Rule

UI must not generate 4K, HDR, DV, Atmos, DTS, TrueHD, or similar badges. Quality badges come from MediaExperienceSnapshot plus ComplianceGateResult.
