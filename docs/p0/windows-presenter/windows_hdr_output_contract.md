# Windows HDR Output Contract

Windows HDR output is a separate contract, not an abstract color decision.

## Required Fields

- display id
- monitor LUID
- GPU adapter id
- advanced color availability
- HDR enabled state
- color space
- swapchain format
- output bit depth
- fullscreen and occlusion state
- multi-monitor state
- OS tone mapping mode
- HDR metadata set result
- display luminance caps
- target display class
- ColorOutputMatrix ref
- output confidence
- optional fallback reason
- trace id
