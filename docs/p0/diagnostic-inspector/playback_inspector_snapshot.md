# Playback Inspector Snapshot

PlaybackInspectorSnapshot is a user and developer explanation contract.

## Required Sections

- user summary
- source summary
- network summary
- cache summary
- video summary
- HDR/DV summary
- audio summary
- subtitle summary
- fallback summary
- quality impact summary
- actions available
- optional developer trace ref
- privacy level
- redaction state

## Rule

Inspector output must explain fallback and decision reasons without leaking credentials, tokens, cookies, or signed URLs.
