# KivoCinemaEngine Conception

KivoCinemaEngine is the Windows private-cinema backend core.

It is not an FFmpeg demo, not an mpv wrapper, not a file browser, and not a UI shell.

The engine chooses the highest reachable experience under source, network, device, display, audio endpoint, subtitle cost, resource budget, and compliance boundaries.

The engine must explain why Direct Play, cache, proxy, fallback, transcode, or reject was selected.

Product difference:

- complex remote sources are first-class
- Direct Play is explainable
- Range, TTL, cache, and fallback are visible
- HDR and DV-aware paths are claim-controlled
- a user-visible playback inspector is part of the core
