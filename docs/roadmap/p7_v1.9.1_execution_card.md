## 16. Stop Conditions

1. Required baseline manifest/hash is missing and not explicitly accepted as a deferral.
2. A HardBlockingGap remains open for the current stage.
3. Public include requires a forbidden dependency.
4. P4 PlaybackRevision ownership is unavailable.
5. P5/P6 async token or completion event has no safe fake fallback.
6. `uint64_t` atomics are not lock-free on the Windows x64 target.
7. CriticalEventMpscRing cannot be implemented without allocation/blocking.
8. Closed-through-Closing cannot be guaranteed.
9. Evidence JSON and execution card disagree.
10. Redline scan finds production pollution.

12. Work would occur outside `C:\kivo video`.
13. Natural family tree is incomplete, missing file responsibilities, or contains dumping-ground directories.
14. Any P7 V1.9.1 requirement, design section, or G1-G60 item is unmapped in the no-omission matrix.
