# GLOBAL DESIGN RULES (V2.0 CONSTANTS)

1. All PTS/PTS/sample position use checked rational rescale — no int64 silent overflow
2. SPSC result = strong typed (Pushed/Full/Closed) — never bool
3. AudioDeviceThread — no throw / no production assert / no heap alloc / no block I/O
4. AudioDataHandle = 3 variants — SyncByteView / OwnedBlock / PoolSlot — no ambiguous span
5. Decode backend has TimedOut state — must close loop
6. PCM decode path ≠ Encoded passthrough parse path — separate backends
7. P6 output contracts = backend-abstract; WASAPI = concrete backend
8. P4 owns playback authority; P6 proposes clock/sync evidence
9. CapabilityRegistry = single P8 claim source
10. EncodedAccessUnit holds OWNED payload only — never borrowed bytes
11. Generation freshness = (epoch, gen) equality — no <= comparison
12. Bit-perfect = 14+ conditions ALL true — any Unknown blocks claim
13. Architecture guard must work in production — not diagnostic-only
14. Benchmarks must include complete environment — no bare numbers
15. Claim boundary cannot be deferred even if licensed production is
